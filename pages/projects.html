Some of my favorite projects

These are the projects I like the most. Some of them I made myself from scratch and others I made while learning from different courses. You can check out more on my GitHub profile.

Read more about me

Forkify App

HTML5 | CSS3 | JS | recipe API

This app allows you to search for the recipe of any food item that you like. Unfortunately the true forkify API is deprecated so the search is pretty limited. The API that is used supports up to 1,000 searches per day.

Budget App

HTML5 | CSS3 | JS

A simple budget app made with vanilla JavaScript. It allows you to calculate your income and expenses with the given input for your current month.

Tech Blog

HTML5 | CSS3 | JS | jQuery

This is a blog I replicated from scratch as a way to test my web developing knowledge of what I've learned so far but also exercise on my UI/UX skills. I added some of my own changes but the project mostly stays true to the original blog.

Pig Game

HTML5 | CSS3 | JS

A small and simple game that makes use of the fundamental commands of vanilla JavaScript. It's played in the browser between 2 players with one or two dice, each player takes turns to rolling the dice and the first to reach 100 points wins.

Fallout4 Pipboy

HTML5 | CSS3 | jQuery | Bootstrap 4

My first project that I completed at the end of a front-end course, everything I have learned throughout the course has been applied here.

Weather App

HTML5 | CSS3 | JS | NodeJS | darksky API | mapbox API | Express | Handlebars

Web based weather app. I created this while following a course focused on back-end development. It uses the forecast API from darksky.net and the geocode API from mapbox.com. I made the website using HTML5, CSS3 and JavaScript with help from other tools such as Express&Handlebars.
